# Installation Guideline

#Install latest version of Xcode

#Clone Repo SSH "git@github.com:BilalKhalid46077/SwiftUI-Employee-Schedule-Management.git"

#Open Xcode go to Click On Open existing projects->Locate repo directory -> open file iOSEmployeeScheduleManagement.xcodeproj

#Xcode will load project

#Click on Play button

#App will run on emulator
